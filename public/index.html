Fire Emblem: Path of Radiance API

Introduction

This is a fan-made project, all the info was gathered by the Fire Emblem community.

This an API that you can make requests to, to get data/stats of the main Characters, the classes, weapons, items and affinities.

Fire Emblem games have a lot of data, and rather than gathering it yourself for your own websites, you can request data to this API, and because it is GraphqQL format you can request only the data that you want in a single query

Instructions

If you got to http://____/graphql, there is a dashboard that you can simulate queries

If you want to make requests to it using fetch or postman, you make a post request to http://____/graphql, the request's body will look consist of a json object, with only on key needed "query", then as the value between quotations you make the request the same way you would do on the dashboard

Example: You want to get a list of all the characters, getting their name, level and base stats

{ "query": "query {characters{name, level, baseStats{hitPoints,strength,magic,skill,speed,luck,defense,resistance}}}" }

Documentation

At the moment, only queries are allowed, no mutations
